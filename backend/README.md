# Backend (Go)

This is the Go backend for the Ethereum Gas & Fee Estimator project.

## Setup

1. Install Go (https://go.dev/doc/install)
2. Install dependencies:
   ```bash
   go mod tidy
   ```
3. Run the server:
   ```bash
   go run main.go
   ```

## Project Structure

- `main.go`: Entry point
- `internal/`: Internal packages and API logic
